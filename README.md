# 🌿 Wijdan Therapy — Fee Sheet Generator

A Streamlit app that generates a branded, personalised, printable PDF fee
sheet — matching the exact colours, layout, and structure of Wijdan Therapy.

## 🚀 Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ Deploy free on Streamlit Cloud

1. Push repo to GitHub
2. Go to share.streamlit.io → New app → pick repo → app.py → Deploy
3. Live in ~60 seconds with a shareable https:// link

## 📁 Files

- app.py            — Streamlit form UI
- pdf_generator.py  — ReportLab PDF engine
- requirements.txt  — streamlit + reportlab

## No API keys needed. Fully offline. No external services.
